### Installation

1. Download the latest `.pext` file from [Releases](https://github.com/BCDezgun/Playnite-HydraTorrent/releases).
2. Open Playnite ‚Üí **Add-ons ‚Üí Install from file...**
3. Restart Playnite.

---

### Setup

1. Go to **Settings ‚Üí Add-ons ‚Üí Hydra Torrent ‚Üí Configure**.
2. Set up your qBittorrent details and test the connection.
3. Add your preferred repack sources (FitGirl and Xatab are highly recommended).
4. Set your default download path (optional).

You're all set! Just click **Hydra Hub** in the sidebar and start searching.
