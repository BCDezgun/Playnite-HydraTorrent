### Installation

1. Download the latest release (`HydraTorrent.pext`) from the **[Releases](https://github.com/YOUR_USERNAME/Playnite-HydraTorrent/releases)** section.
2. In Playnite, go to **Menu ‚Üí Add-ons ‚Üí Install from file...**
3. Select the downloaded `.pext` file.
4. Restart Playnite.

---

### Configuration

1. Open **Settings ‚Üí Plugins ‚Üí HydraTorrent ‚Üí Configure**.
2. Enable qBittorrent and provide:
   - Address: `127.0.0.1`
   - Port: `8080` (default)
   - Login and Password for the qBittorrent WebUI.
3. Click **"Test Connection"** ‚Äî it should show ‚úÖ.
4. (Optional) Set a default installation path for your games.

---

### How to Use

1. Click the **üîç Torrent Search (Hydra)** icon in the sidebar.
2. Type the game title ‚Üí find the desired repack.
3. Double-click ‚Üí edit the title if needed ‚Üí add to library.
4. Go to the game card ‚Üí click **"Install"**.
5. Choose a path (or use the default) ‚Üí the torrent will be sent to qBittorrent immediately!
